## string 객체는 문자열 입력 받을 때 몇 자를 입력할지 물어 보지않고 알아서 크기를 자동 조절한다. 동작 원리를 검색해보고 설명하라.
- std::string은 size와 capacity를 관리하며, 문자열이 capacity를 넘으면 자동으로 메모리를 2배 정도 늘려 재할당한다. 짧은 문자열은 SSO로 객체 내부에 저장한다.그래서 입력받을 때 크기를 미리 정하지 않아도 알아서 조절된다.
## 문자열을 입력할 때 마지막에 입력하는 엔터키와 getline 함수의 마지막 인자에 전달하는 구분문자와 차이를 설명하라.
- 사용자가 입력할 때 누르는 엔터키는 입력 버퍼에 \n를 실제로 넣는 신호이고, getline의 구분자는 읽기를 어디서 멈출지 정하는 문자(기본값 \n)입니다. getline은 이 구분자를 만나면 그 문자를 string에 저장하지 않고 버린 채로 읽기를 종료합니다. 따라서 엔터키를 누르면 \n을 만나 기본적으로 입력이 끝나지만, 구분자를 다른 문자로 바꾸면 그 문자에서 멈춥니다.
