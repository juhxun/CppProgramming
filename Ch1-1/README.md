### STL(Standard Template Library) 라이브러리

- 여러 자료 구조, 함수, 알고리즘 등을 쓰기 쉽게 정형화해서 라이브러리화 해둔 것이다.
라이브러리란?
프로그램에서 사용할 수 있도록 미리 만들어져 있는 함수와 변수들의 모음집이다. 
예를 들면 C에서 쓴 <stdio.h> 라이브러리에는 printf, scanf등 표준 입출력 함수에 대한 것들이 담겨있고, 
다른 라이브러리로는 <iostream>, <string> 등이 있다.

STL은 C++을 위한 라이브러리로  알고리즘, 컨테이너, 함수자, 반복자 4가지로 구성되어 있다. 
여기서 가장 큰 것을 차지하는 것은 컨테이너이다.

STL Container 컨테이너
흔히 자료 구조라 하며, 데이터를 저장하는 객체이다. 

STL Iterator 반복자
반복자는 포인터와 비슷한 개념으로, 각 컨테이너의 데이터 위치를 가리킨다.

STL Algorithm 알고리즘
자주 사용하며 유용한 알고리즘 일부가 STL로 제공되며, 그 예로 sort, binary_search, lower_bound 등 정렬, 검색, 연산 등을 해결하는 함수가 있다.

STL Function Object 함수자
STL은 함수 호출 연산자 operator()를 오버로드하는 클래스들을 포함한다.
