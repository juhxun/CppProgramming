### STL(Standard Template Library) 라이브러리란?

STL(Standard Template Library)은 C++에서 제공하는 표준 템플릿 기반 라이브러리로,  
자주 사용되는 자료구조와 알고리즘을 미리 구현해 둔 라이브러리이다.

STL을 사용하면 자료구조와 알고리즘을 직접 구현하지 않아도 되어  
개발 속도와 코드의 안정성을 크게 향상시킬 수 있다.
