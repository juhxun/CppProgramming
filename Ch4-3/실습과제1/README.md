# 실습과제1

## 메모리 누수를 설명하라.
- `new`로 메모리를 할당하고 `Delete`를 호출하지 않아 프로그램이 끝날 때 까지 메모리가 해제되지 않는 현상이다.
## 메모리 누수가 왜 심각한 문제를 발생시키는지 설명하라.
- 메모리가 해제되지 않기 때문에 프로그램이 오래 실행될 수록 메모리 사용량이 증가하고 성능 저하 및 크래시가 발생한다.
## 메모리 누수를 방지하는 방법을 설명하라.
- `Delete` 사용이 가장 기본적인 방법이며 스마트 포인터를 사용해 방지 할 수 있다.
## C++ 스마트 포인터 클래스에 대해 조사하라.
- C++에서 메모리를 자동으로 관리해주는 포인터 객체로 `new / delete`를 직접 안 써도 되게 만들어주는 도구이다.
