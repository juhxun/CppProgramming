## 전처리 또는 선행처리 단계를 설명하라
- 전처리는 컴파일 이전 단계에서 수행되며, # 기호로 시작하는 전처리 지시문을 처리합니다.
## 객체 cin, cout은 어디에 선언되어 있는가? 
    선언된 파일을 찾아서의미를 설명하고 캡쳐하여 첨부하라
- `<iostream>` 헤더파일의 `std namespace` 안에 선언 되어있다.

## 프로그램을 링크할때 이름 충돌이 발생하는 경우를 설명해보라
- 소스 파일에서 동일한 이름의 함수나 변수가 정의되어 있을 때 발생합니다.
## 스트림이란 무엇인가?
- 스트림은 프로그램과 입출력 장치 사이에서 데이터가 흐르는 통로를 의미합니다.
## 표준 출력 스트림을 정의할 때 사용하는 C언어 자료형은?
- `file` 자료형을 사용한다.
## 표준 출력 스트림을 정의할 때 사용하는 C++언어 자료형은
- `ostream` 자료형을 사용한다.
