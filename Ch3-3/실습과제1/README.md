# 실습과제 1

## 객체의 생성과 소멸의 의미를 설명하라
- 객체의 생성은 클래스를 바탕으로 실제 메모리에 객체를 생성하는 것이고 객체의 소멸은 다 사용한 객체를 메모리에서 지워주는 것입니다.
---
## 지역변수(지역객체), 전역변수(전역객체)의 차이를 조사하라, 선언위치, 접근범위, 생성시점, 소멸시점, 초기화 여부 등의 관점에서 비교하라
|-| 지역변수(지역객체) | 전역변수(전역객체)|
|-|:--------------------:|:------------------:|
|선언위치|함수 내부|함수 밖|
|접근범위|선언된 함수 또는 내부에서만 접근 가능|전체 접근 가능|
|생성시점|함수가 생성될 때|프로그램이 사작할 때|
|소멸시점|함수 또는 프로그램이 종료될 때|프로그램이 종료 될 때|

---
## 프로그램 실행 후 main함수가 실행되기 전에 어떤 일이 처리되는지 조사해보라
- 메모리 할당 -> 전역 변수 초기화 - > 라이브러리 설정 후 메인 함수 호출준비
---
## main함수가 종료 후 프로그램이 종료하기 전에 어떤 일이 처리되는지 조사하라
- 반환값 처리 -> 종류 함수 실행 -> 전역 객체 소멸 -> 프로그램 종료
