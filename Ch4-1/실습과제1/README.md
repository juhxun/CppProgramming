# 배열명의 의미를 설명하라.
- 배열을 구별하는 이름으로 배열명은 배열의 시작 주소를 나타내며, 이를 통해 인덱스를 사용하여 각 요소에 접근 할 수 있다.
# 객체 포인터의 의미를 설명하라.
- `객체의 주소`를 저장하는 변수이며, **객체가 아님** 그리고 객체 포인터 선언은 `클래스명* 객체포인터명;`으로 할 수 있다.
# 포인터를 이용하여 배열 각 원소의 주소를 구하는 방법을 설명하라.
- 포인터 연산을 이용하면 각 원소의 주소를 쉽게 구할 수 있습니다.
# 포인터를 이용하여 배열 각 원소의 값을 구하는 방법을 설명하라.
- 포인터를 이용해 해당 주소에 저장된 값을 얻으려면 `역참조 연산자(*)`를 사용합니다.
