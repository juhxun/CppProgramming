# 실습과제 1

## 인자와 매개변수를 구분하여 설명하시오.
- 인자는 함수를 호출할 때 넘겨주는 값이고 매개변수는 함수 정의에서 값을 받기 위해 선언된 변수임.
- ## 함수 호출시 자동으로 처리되는 2가지는 무엇인가?
- 1. 인자의 값이 매개변수로 전달됨. 2. 복귀 주소가 스택에 저장됨.
## 값에 의한 호출로 다른 함수에서 선언된 변수의 값을 변경할수 없다. 이유를 설명하라.
- 함수에 전달되는 것은 원본이 아닌 복사본이며, 함수 내부에서 값을 바꿔도 복사된 값만 변경됨.
## 주소에 의한 호출을 사용해야하는 경우를 2가지 이상 설명하라.
- 1. 함수에서 원본 값을 직접 변경해야 할 때. 2. 여러 값을 동시에 반환하고 싶을 때.
## 인자가 객체인 경우는 어떤 호출방식이 좋은가? 이유도 설명하라.
- 객체는 크기가 클 수 있음이며, 값 복사 시 성능 저하 발생함. 참조`(&)`를 쓰면 복사 없이 원본 사용 가능함.
## 함수 반환과정에서 자동으로 처리되는 2가지는 무엇인가?
- 1. 반환값이 호출한 곳으로 전달됨. 2. 저장된 복귀 주소로 처리됨.
## 함수 반환과정에서 임시변수가 필요한 이유를 설명하라.
- 반환값을 안전하게 전달하기 위해서 함수 내부의 지역변수는 함수 종료 시 소멸됨, 반환 시 값을 임시 공간에 복사해서 전달해야 함.
