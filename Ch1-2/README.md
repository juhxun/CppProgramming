## IDE란 무엇인가?

- IDE는 코드 작성, 실행, 디버깅, 빌드 등 **개발에 필요한 기능을 하나로 통합한 프로그램**이다.

---

## IDE의 종류

**범용적인 IDE**
- Visual Studio : C#, C++ 등등 개발
- VS Code : 가볍고 확장 가능

**언어/플랫폼 특화된 IDE**
- PyCharm : 파이썬 개발
- Android Studio : 안드로이드 앱 개발

---

## IDE에 포함된 기능

- 코드 편집기 : 문법 강조, 자동 완성, 코드 포맷팅
- 컴파일러/인터프리터 : 코드 실행
- 디버거 : 중단점 설정, 변수 확인
- 빌드 자동화 도구 : 프로젝트 빌드 및 패키지 관리
- 버전 관리 연동 : Git 사용 가능
- 프로젝트 관리 : 파일/폴더 구조, 라이브러리 관리
