# 실습과제2
## 매크로상수 NULL이 무엇인지 조사하라, C/C++언어에서 정의의 차이, 정의된 헤더파일, 사용용도 등
| 구분      | C                 | C++ |
| ------- | ----------------- | --- |
| 정의      | `(void*)0` 또는 `0` | `0` |
| 타입      | void 포인터          | 정수  |
| 타입 안정성  | 낮음                | 낮음  |
| 오버로딩 문제 | 없음                | 있음  |

## 널문자와 NULL의 차이를 설명하시오, 정의, 자료형, 메모리크기, 실제값, 사용용도 등
| 항목    | 널 문자          | NULL                |
| ------- | ---------------- | --------------------|
| 의미     | 문자열 끝 표시    | 유효하지 않은 포인터 |
| 사용 대상| 문자/문자열       | 포인터              |
| 자료형   | char             | 포인터 상수         |
| 값       | 0 (문자 코드)     | 0 (주소)           |
| 크기     | 1 byte           | 4/8 byte           |
| 사용 위치 | 문자열 내부      | 포인터 변수         |

## 요즘은 NULL 대신에 nullptr을 사용하는데 nullptr은 무엇인지 조사하시오
| 구분         | NULL           | nullptr        |
| -------------| -------------------| -------------- |
| 도입          | C / C++            | C++11          |
| 실제 값       | 0                  | 특별한 키워드        |
| 타입          |  int (또는 void*)  | std::nullptr_t |
| 포인터 변환   | 가능                | 가능             |
| 정수 변환     | 가능                | ❌ 불가능          |
| 오버로딩 문제 | 발생                | 없음             |
