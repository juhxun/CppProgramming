# 실습과제 1
## C++에서 생성자를 통하여 멤버변수의 초기화를 하는 이유를 설명하라.
- 객체 생성과 동시에 멤버를 유효한 상태로 만든다.
---
## 생성자 작성시 생성자 몸체에서 멤버를 초기화하는 것 보다 멤버 초기화 리스트에서 초기화하는 이유를 설명하라.
- 멤버를 생성 시점에 직접 초기화하여 불필요한 초기화/대입을 방지할 수 있다.
---
## 위임, 타겟 생성자로 나누어 작성하는 이유를 설명하라.
- 초기화 코드 중복을 제거하고 유지보수성을 높일수 있다.
---
## 함수 몸체(body)에서 생성자를 직접 호출하면 어떻게 되는지조사해보라.
- 현재 객체는 초기화되지 않고 임시 객체만 생성되었다가 사라진다.
